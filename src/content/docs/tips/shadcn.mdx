<Steps>
1. Install react using your preferred package manager:
    <Tabs syncKey="shadcn">
      <TabItem label="npm" icon="seti:npm">
        ```sh
        npx astro add react
        ```
      </TabItem>
      <TabItem label="pnpm" icon="pnpm">
        ```sh
        pnpx astro add react
        ```
      </TabItem>
      <TabItem label="yarn" icon="seti:yarn">
        ```sh
        npx astro add react
        ```
      </TabItem>
      <TabItem label="bun" icon="bun">
        ```sh
        bunx astro add react
        ```
      </TabItem>
    </Tabs>
2. Add Tailwind CSS to your project
    <Tabs syncKey="shadcn">
      <TabItem label="npm" icon="seti:npm">
        ```sh
        npx astro add tailwind
        ```
      </TabItem>
      <TabItem label="pnpm" icon="pnpm">
        ```sh
        pnpx astro add tailwind
        ```
      </TabItem>
      <TabItem label="yarn" icon="seti:yarn">
        ```sh
        npx astro add tailwind
        ```
      </TabItem>
      <TabItem label="bun" icon="bun">
        ```sh
        bunx astro add tailwind
        ```
      </TabItem>
    </Tabs>
3. Create a `styles/globals.css` file in your `src/pages/index.astro` file

        ```css title="src/styles/globals.css"
        @tailwind base;
        @tailwind components;
        @tailwind utilities;
        ```
4. Import the `styles/globals.css` file in the `src/pages/index.astro` file:

        ```astro title="src/pages/index.astro"
        ---
        import '@/styles/globals.css'
        ---
        ```
5. Update `astro.config.mjs` and set `applyBaseStyles` to `false`:

        ```js title="astro.config.mjs"
            export default defineConfig({
            integrations: [
                tailwind({
                applyBaseStyles: false,
                }),
            ],
        });
        ```
6. Edit tsconfig.json file
    Add the following code to the `tsconfig.json` file to resolve paths:
    
        ```ts title="tsconfig.json" {4-9} showLineNumbers
        {
            "compilerOptions": {
                // ...
                "baseUrl": ".",
                "paths": {
                "@/*": [
                    "./src/*"
                ]
                }
            }
        }
    ```

7. Run the CLI
    <Tabs syncKey="shadcn">
      <TabItem label="npm" icon="seti:npm">
        ```sh
       npx shadcn@latest init
        ```
      </TabItem>
      <TabItem label="pnpm" icon="pnpm">
        ```sh
        pnpx shadcn@latest init
        ```
      </TabItem>
      <TabItem label="yarn" icon="seti:yarn">
        ```sh
        npx shadcn@latest init
        ```
      </TabItem>
      <TabItem label="bun" icon="bun">
        ```sh
        bunx shadcn@latest init
        ```
      </TabItem>
    </Tabs>
10. That's it, you're all set up to use Shadcn UI in your project!
    You can now start adding components from the Shadcn UI library to your project.
    
        <Tabs syncKey="shadcn">
      <TabItem label="npm" icon="seti:npm">
        ```sh
       npx shadcn@latest add button
        ```
      </TabItem>
      <TabItem label="pnpm" icon="pnpm">
        ```sh
        pnpx shadcn@latest add button
        ```
      </TabItem>
      <TabItem label="yarn" icon="seti:yarn">
        ```sh
        npx shadcn@latest add button
        ```
      </TabItem>
      <TabItem label="bun" icon="bun">
        ```sh
        bunx shadcn@latest add button
        ```
      </TabItem>
    </Tabs>

        The command above will add the `Button` component to your project. You can then import it like this:

        ```astro title="src/pages/index.astro" {2,10}
        ---
        import { Button } from "@/components/ui/button"
        ---

        <html lang="en">
        <head>
            <title>Astro</title>
        </head>
        <body>
            <Button>Hello World</Button>
        </body>
        </html>
        ```
</Steps>

## Working with Interactive Components

<Aside>
If you getting an error when trying to use interactive components, you will need to wrap each component in a `.tsx` file.
</Aside>

<Steps>
1. Add the Drawer component to your project:

    <Tabs syncKey="shadcn">
      <TabItem label="npm" icon="seti:npm">
        ```sh
       npx shadcn@latest add drawer
        ```
      </TabItem>
      <TabItem label="pnpm" icon="pnpm">
        ```sh
        pnpx shadcn@latest add drawer
        ```
      </TabItem>
      <TabItem label="yarn" icon="seti:yarn">
        ```sh
        npx shadcn@latest add drawer
        ```
      </TabItem>
      <TabItem label="bun" icon="bun">
        ```sh
        bunx shadcn@latest add drawer
        ```
      </TabItem>
    </Tabs>

2. Create a `Drawer` Component in your project:

    ```tsx title="src/components/MyDrawer.tsx"
    import { Button } from "@/components/ui/button";
    import {
        Drawer,
        DrawerClose,
        DrawerContent,
        DrawerDescription,
        DrawerFooter,
        DrawerHeader,
        DrawerTitle,
        DrawerTrigger,
    } from "@/components/ui/drawer";

    export const MyDrawer = () => {
        return (
            <Drawer>
                <DrawerTrigger>Open</DrawerTrigger>
                <DrawerContent>
                    <DrawerHeader>
                        <DrawerTitle>Are you absolutely sure?</DrawerTitle>
                        <DrawerDescription>This action cannot be undone.</DrawerDescription>
                    </DrawerHeader>
                    <DrawerFooter>
                        <Button>Submit</Button>
                        <DrawerClose>
                            <Button variant="outline">Cancel</Button>
                        </DrawerClose>
                    </DrawerFooter>
                </DrawerContent>
            </Drawer>
        );
    };
    ```
3. Import the `MyDrawer` component in the `src/pages/index.astro` file:

    ```astro title="src/pages/index.astro" {2,10}
    ---
    import { Button } from "@/components/ui/button"
    import { MyDrawer } from "@/components/MyDrawer"
    import '@/styles/globals.css'
    ---

    <html lang="en">
    <head>
        <title>Astro</title>
    </head>
    <body>
        <Button>Hello World</Button>
        <MyDrawer client:load />
    </body>
    </html>
    ```
</Steps>
