---
title: How to add GSAP to your Astro
---

import { Tabs, TabItem, LinkCard } from "@astrojs/starlight/components";

You can easily add GSAP to your Astro project!

To start, install GSAP.

<Tabs>
    <TabItem label="npm">
        ```sh
            npm install gsap
        ```
    </TabItem>
    <TabItem label="pnpm">
        ```sh
            pnpm add gsap
        ```
    </TabItem>
    <TabItem label="yarn">
        ```sh
            yarn add gsap
        ```
    </TabItem>
</Tabs>

After it's installed you can import it into your project inside a `script` tag on your page.

```html title="MyComponent.astro"
<script>
import { gsap } from "gsap";

const box = document.getElementById("box");
const myTimeline = gsap.timeline();

myTimeline
    .from(box, {
        opacity: 0,
    })
    .to(box, {
        opacity: 1,
        duration: 0.25,
    })
</script>

<div id="box"></div>
```

And that's it, you're all set up to use GSAP in your project!

To learn more about GSAP, check their [official site](https://www.gsap.com)!
